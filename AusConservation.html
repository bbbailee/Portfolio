Process

1

Research

This includes topic research, who would be interested in the topic of the website, design trends and current web designs.

2

Ideating

Brainstorming ways of displaying information and engaging users in educational content.

3

Designing concepts

This includes colour palette and typographical testing, concept mood boards, layout testing and creating mock up designs of each concept.

4

Final Design

Using the chosen concept to design a website using HTML, CSS and JavaScript. I also used animations in the design of this website, which involved using libraries such as GSAP.
