Pools & Falls

A website that showcases the best swimming locations in Australia.

Website

Brief

A website that will list places around Australia that people can visit and enjoy a swim.

Solution

A website that helps users discover new locations they haven't heard of, and be prepared for visiting these locations with all the basic information they need.

Process

1

Research

Researching the topic, looking at relevant sources on locations around the country and looking at existing websites that have the same/similar topic and functionalities for inspiration on how to approach planning my website.

2

Ideating

Ideating for this project looked like brainstorming, and creating rough sketches of design layouts.

3

Designing concepts

Concept designing for this project included using Figma to piece together information and images, colour palette testing, typographical testing, layout testing and creating device mockups to visualise the final design. Concept sketching and getting user and peer feedback were also essential to this part of the process.

4

Final Design

Creating the final website involved coding HTML, CSS and JavaScript, and implementing and user feedback.

Tools

For this project I used Figma to create mock ups for the web and mobile designs. I also used HTML, CSS and JavaScript to code the website.
